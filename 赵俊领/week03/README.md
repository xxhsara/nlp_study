### regex模型：
#### 优点：
- 1.正则匹配，规则明确
- 2.不需要训练数据
- 3.准确率高，无计算要求
#### 缺点：
- 1.范围有限，适合固定格式的数据
- 2.规则不灵活，只能表面匹配
- 3.规则未覆盖的会归为Other


### tfidf模型：
#### 优点：
- 1.关键词匹配，规则灵活
- 2.推理速度快，资源消耗低
- 3.模型文件小

#### 缺点：
- 1.依赖分词质量，受jieba分词精度影响
- 2.对同义词，多义词处理不佳


### bert模型：
#### 优点：
- 1.基于Transformer架构，能捕捉深层次的语义关系
- 2.在大规模中文语料上预训练，具备丰富的语言知识
- 3.端到端处理，无需分词
- 4.准确率高

#### 缺点：
- 1.模型体积大，资源消耗大
- 2.训练速度慢，耗时长
- 3.模型效果差，需要调参

### LLM模型：
#### 优点：
- 1.基于大模型预训练，知识丰富，泛化能力强
- 2.样本少，无需大量标注数据
- 3.可根据提示词改变行为

#### 缺点：
- 1.速度慢，API调用有延迟
- 2.成本高
- 3.结果不稳定，受提示词影响
