# 1.BERT
### 优点：
1.先训练没有标注的语料库，然后微调

2.分类判断比较强

### 缺点：
1.占内存，慢

2.只能理解，不能生成新的内容

# 2.Prompt
### 优点：
1.场景少，训练数据少也可以训练

2.只修改提示词就可以，方便一些

### 缺点：
1.比较依赖提示词

2.小模型可能不准确

3.不适合分类任务

# 3.Regex_rule
### 优点：
1.快速

2.按照自己的规则判断，自定义属性强

### 缺点：
1.误判

2.不会理解

# 4.Tfidf_ml
### 优点：
1.适合文本

2.训练快，部署快

### 缺点：
1.这个区分不了长难句

2.不会以此类推
